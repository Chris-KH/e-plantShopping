# Paradise Nursery Shopping App

Paradise Nursery is a React + TypeScript shopping interface for indoor plants. The app includes:

- A landing page with brand story and a Get Started call-to-action
- A categorized plant product listing page
- A shopping cart page with quantity controls, delete actions, and running totals

## Tech Stack

- React 19
- TypeScript
- React Router
- Redux Toolkit + React Redux
- Vite

## Run Locally

1. Install dependencies:

```bash
npm install
```

1. Start development server:

```bash
npm run dev
```

1. Build production bundle:

```bash
npm run build
```
